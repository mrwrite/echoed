## ADDED Requirements

### Requirement: Academic lesson structure does not replace lesson delivery
The existing lesson delivery and activity system SHALL continue to operate through the current Course -> Unit -> Lesson -> Activity flow while rendering academic lesson structure fields alongside activities.

#### Scenario: Lesson contains instructional structure and activities
- **WHEN** a lesson includes both academic structure fields and lesson activities
- **THEN** the instructional sections render without replacing or duplicating the existing activity flow
