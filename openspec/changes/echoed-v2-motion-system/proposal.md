# EchoEd V2 Motion System

## Why

EchoEd V2 now has a premium visual language. It needs a consistent motion layer so interaction and page transitions feel smooth, confident, fast, and intentional without introducing playful or distracting effects.

## What Changes

- Add global motion tokens and reusable CSS utility classes.
- Add reduced-motion handling for transform, shimmer, and skeleton motion.
- Apply CSS-only motion polish to V2 marketing/auth pages, workspace dashboards, Product Studio, Review Center, Learner Portal, Analytics, Projects, and Artifacts.

## Non-Goals

- No backend changes.
- No route changes.
- No authentication changes.
- No course, lesson, learner, progress, governance, product, access, or analytics behavior changes.
- No new product features, AI execution, payments, or commercialization logic.
