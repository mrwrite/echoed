## ADDED Requirements

### Requirement: Canonical flagship activity pattern set
The system SHALL implement a bounded set of canonical flagship activity patterns including interactive reflection, guided discussion, map/geography interaction, ecosystem exploration, storytelling interaction, vocabulary reinforcement, multimedia exploration, lightweight project activity, and culturally grounded exploration prompts.

#### Scenario: Rich activities stay inside one canonical system
- **WHEN** rich flagship activities are added
- **THEN** they MUST be implemented as bounded canonical patterns rather than disconnected mini-games or a separate activity engine

### Requirement: Activity pattern reuse and restraint
The flagship implementation SHALL avoid activity-type explosion by reusing existing lesson/activity foundations where possible and adding only the minimum new patterns needed for the canonical pathway experience.

#### Scenario: Activity growth remains governable
- **WHEN** new learner activity shapes are proposed for the flagship pathway
- **THEN** they MUST be justified as reusable patterns and MUST NOT create fragmented activity paradigms
