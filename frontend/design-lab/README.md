# EchoEd UX Overhaul Design Lab

This is an isolated static prototype for the role-based UX overhaul design phase.

It does not affect Angular production routing, services, auth, or backend contracts.

Open `frontend/design-lab/index.html` in a browser to review representative screens:

- Public landing
- Login
- Student dashboard
- Student course overview
- Student lesson player
- Teacher dashboard
- Teacher class detail
- Teacher assignment workflow
- Admin dashboard
- Admin user management
- Shared states
- Desktop and mobile navigation concepts
