# EchoEd V2 Landing Redesign

## Why

EchoEd needs a premium homepage that presents the V2 platform clearly to investors, schools, enterprises, parents, and teachers. The landing page should match the approved premium visual language while preserving existing navigation and authentication entry points.

## What Changes

- Replace the current landing page presentation with a premium marketing experience.
- Add a large editorial hero, glass platform preview panel, animated cards, feature grid, demo section, roadmap section, testimonials, walkthrough, and final CTA.
- Preserve existing route targets for sign in, account creation, and demo launch.

## Non-Goals

- No backend changes.
- No routing changes.
- No API changes.
- No authentication changes.
- No application behavior changes beyond landing-page presentation.
