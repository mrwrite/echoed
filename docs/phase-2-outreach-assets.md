# Phase 2 Outreach Assets

These templates are for careful, organic outreach. Do not mass-send them. Personalize each message and keep the ask bounded.

## Shared Language

Use this short project description when helpful:

```text
EchoEd is an early open-source education platform centered on African and African-American history. It has a working K-5 demo and is looking for bounded educator feedback, contributor help, and community guidance before broader outreach.
```

Use this safety note when linking the demo:

```text
The demo uses shared demo-only credentials. Please do not enter personal, student, school, or production data.
```
