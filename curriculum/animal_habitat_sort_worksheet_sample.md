# Animal Habitat Sort Worksheet (Sample)

## Match the Animal to Its Habitat

**Name:** ____________________  
**Date:** ____________________

### Directions
Draw a line from each animal to the place it lives best.

| Animal | Habitat Choices |
|---|---|
| Lion | Savanna / Rainforest / Desert |
| Gorilla | Rainforest / Savanna / River |
| Camel | Desert / Rainforest / Wetland |
| Hippopotamus | River/Wetland / Desert / Mountain |

### Challenge (Grades 3–5)
Write one complete sentence:

> The __________________ lives in the __________________ because __________________.

### Draw It
Draw your favorite African animal in its habitat. Label **2 habitat details** (example: grass, trees, water, sand).

---

## Teacher Key
- Lion → Savanna
- Gorilla → Rainforest
- Camel → Desert
- Hippopotamus → River/Wetland
